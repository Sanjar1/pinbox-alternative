## 1) Yandex Business API / Partner Access

Subject: Request API/Partner Access for Multi-Branch Listing Management

Hello Yandex Business Team,

We manage a chain with approximately 36-40 locations and need official API/partner access for centralized listing management.

Use case:
- One admin dashboard maintains canonical store data.
- We need to sync listing updates for multiple branches.
- We require compliant, auditable update workflows.

Please confirm whether our account can access operations for:
1. Listing read/search
2. Listing update (name, address, pin, phone, hours, website, description)
3. Listing creation
4. Listing close/deactivate

Please also provide:
- Required auth model (OAuth/token/partner account)
- API documentation link for our approved access mode
- Rate limits and moderation timelines
- Any onboarding/compliance requirements

Thank you.
